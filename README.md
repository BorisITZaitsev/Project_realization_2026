# АИС «Налоговая инспекция»

Автоматизированная информационная система учёта юридических лиц и видов их деятельности (ОКВЭД).

## Требования

- **Python 3.8+** — только стандартные библиотеки (tkinter, sqlite3)
- Дополнительных пакетов **не требуется**

## Запуск

### Windows
Дважды щёлкните `Запустить.bat`  
или:
```
cd src
python main.py
```

### Linux / macOS
```bash
cd src
python3 main.py
```

## Функциональность

### Юридические лица
- Полный CRUD с валидацией всех полей (ИНН 10/12 цифр, КПП 9 цифр, email, дата)
- Архивное удаление (физически не удаляются)
- Поиск по ИНН, наименованию, адресу (любое поле или все сразу)
- Флаг «есть задолженность» — красная подсветка строки
- Двух-вкладочная форма: «Реквизиты» + «Банк»
- Вкладка ОКВЭД прямо в форме редактирования
- Печать карточки организации

### Справочник ОКВЭД
- Добавление / редактирование / архивирование кодов
- Отображение количества ЮрЛиц по каждому коду
- Связь «многие ко многим»: одна организация — несколько кодов ОКВЭД

### Поиск по ОКВЭД
- Кнопка «📋 По ОКВЭД…» на панели ЮрЛиц
- Выбор кода → мгновенный список организаций с этим кодом
- Печать списка в браузере

### Отчёты (HTML → браузер → кнопка «Печать»)
| Отчёт | Описание |
|-------|----------|
| Ведомость задолженников | Все ЮрЛица с флагом задолженности |
| Карточка ЮрЛица | Все реквизиты + список ОКВЭД |
| Список по ОКВЭД | Организации с выбранным кодом |
| Статистика по ОКВЭД | Кол-во ЮрЛиц по каждому коду |

### Разграничение прав доступа (2 роли)
| Действие | Оператор | Администратор |
|----------|----------|---------------|
| Просмотр / поиск | ✅ | ✅ |
| Добавление / редактирование ЮрЛиц | ✅ | ✅ |
| Управление справочником ОКВЭД | ❌ | ✅ |
| Управление пользователями | ❌ | ✅ |
| Резервное копирование | ❌ | ✅ |

### Безопасность
- Пароли хранятся в виде SHA-256 хэша
- Вход через диалоговое окно при каждом запуске
- Блокировка функций по роли
